# network-latency-logger

## Internet Connection Monitor

A small legacy GUI tool for monitoring internet connection stability over a specified period of time.

The application allows the user to set the monitoring duration and ping interval in seconds. During execution, it sends ping requests to Google DNS servers and saves the ping results to an output file.

## Features

- Simple GUI for entering monitoring settings
- Configurable test duration
- Configurable ping interval
- Sends ping requests to check connection stability
- Saves ping results to a file

## Project Status

This is an old personal project created for practical use. It is not intended to be a professional or production-ready tool, but is kept here as a simple example of GUI input handling, basic network checking, automation, and file output.
